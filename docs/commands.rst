Command Reference
=================

Basic Commands
--------------

- ``--ip`` – Show current IP
- ``--change`` – Change IP once
- ``--interval`` – Interval between IP changes
- ``--count`` – Number of changes
- ``--stop`` – Stop Tor services
- ``--version`` – Show version

Advanced Commands
-----------------

- ``--status`` – Show system status
- ``--country`` – Select exit country
- ``--schedule`` – Scheduled rotation
- ``--dns-leak-test`` – DNS leak test
- ``--kill-switch`` – Toggle kill switch
- ``--log`` – View logs
- ``--follow`` – Follow logs
- ``--json`` – JSON output
- ``--auto-fix`` – Auto dependency install
